# Hospital Document Management System

## Project Description
This project is a front-end web application that simulates a hospital document management system. It was developed using HTML, CSS, and JavaScript.

## Features
- User login using sessionStorage
- Add and delete patient documents
- Data persistence using localStorage
- Load external data using Fetch API (AJAX)
- Simple dashboard interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- Web Storage API
- Fetch API

## How to Run
1. Open login.html
2. Enter a username
3. Access dashboard
